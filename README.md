# Deep-Focus

Monorepo scaffold for the Deep-Focus Chrome extension and FastAPI backend.

## Structure

```text
deep-focus/
├── extension/
└── backend/
```

## Apps

- `extension`: Vite + React + TypeScript + Tailwind CSS, prepared for Chrome Manifest V3.
- `backend`: FastAPI + Uvicorn + Pydantic with a clean app structure.

Each app can be installed and run independently from its own folder.
