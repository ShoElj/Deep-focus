# Deep-Focus Extension

Chrome Manifest V3 extension scaffold built with Vite, React, TypeScript, and Tailwind CSS.

## Install

```bash
npm install
```

## Run

```bash
npm run dev
```

## Build

```bash
npm run build
```

Load the generated `dist/` directory in Chrome as an unpacked extension after building:

1. Open `chrome://extensions`.
2. Enable Developer mode.
3. Select Load unpacked.
4. Choose `extension/dist`.
